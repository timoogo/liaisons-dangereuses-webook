# Symfony of the galaxy

## installation

*NEED COMPOSER AND SYMFONY*
```bash
git clone git@github.com:IIM-Creative-Technology/symfony-timoogo.git
```
then
````bash
cd symfony-timoogo
````
then
```bash
composer install
```
then
```bash
npm install
```
